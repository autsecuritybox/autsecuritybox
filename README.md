## `whoami`

**AutSecurity Box** es una plataforma de aprendizaje en ciberseguridad ofensiva orientada a la comunidad hispanohablante. Aquí encontrarás todo lo que uso y publico: herramientas, entornos preconfigurados, writeups, scripts, documentación y recursos para desarrollar habilidades reales en hacking ético y CTF.

```
  Plataforma : box.autsecurity.com
  Enfoque    : Offensive Security · CTF · Red Team
  Entorno    : Kali Linux + bspwm + Kitty + Polybar
  Idioma     : Español (con recursos en inglés)
  Filosofía  : "El conocimiento que no se comparte es conocimiento que se pierde."
```

### 🛠️ [AutSecBox — Kali Setup](https://github.com/autsecuritybox/autsecbox-kaliup)

Entorno profesional de pentesting y CTF. Dotfiles reales con un script de aprovisionamiento para cualquier Kali Linux limpio.

```bash
git clone https://github.com/autsecuritybox/autsecbox-kaliup
cd autsecbox-kaliup && ./setup.sh
```

**Incluye:**
- `bspwm` · `Kitty` · `Polybar` · `Powerlevel10k`
- CVE-2026 carousel en la barra (NVD API)
- Portapapeles dual (F1-F4) · Scope status
- Colores, fuentes y shortcuts optimizados para CTF
